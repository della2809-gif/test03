# 건강자산 프리미엄 코칭 MVP

사전 선별 문진과 현장 인바디·혈압 측정, 조건부 정밀 문진을 연결하는 한국어 건강 코칭 워크스페이스입니다. 의료 진단이나 처방을 제공하지 않으며 점수는 규칙 기반 웰니스 지표입니다.

## 핵심 흐름

1. 방문 전 핵심 선별 문진 36개
2. 현장 인바디·허리둘레·혈압 측정
3. 신호가 관찰된 축만 2~4개의 정밀 문진
4. 규칙 기반 축 점수, 신뢰도 및 우선순위 계산
5. 코치가 임시 리포트를 확인하고 승인
6. 4주 목표와 추적 계획 발행

## 실행

Node.js 22.13 이상이 필요합니다.

```bash
npm ci
npm run dev
```

검증은 `npm test`로 실행합니다. 민감 건강정보를 운영 환경에서 저장하려면 인증, 동의, 접근 로그와 행 단위 접근 제어 정책을 먼저 활성화해야 합니다.

## GitHub Pages 배포

`main` 브랜치에 변경사항을 푸시하면 `.github/workflows/deploy-pages.yml`이 정적 사이트를 빌드해 GitHub Pages로 자동 배포합니다.
